Figure Caption Extraction App

Project Overview

Figure Caption Extraction and Access System is a production-ready tool designed to help researchers and developers extract, analyze, and query structured metadata from scientific papers, with a focus on figure captions and biological entities.

How It Works
Data Ingestion: Fetches paper metadata (title, abstract), figure captions, and image URLs from PubMed Central (PMC) and PubTator APIs.
Processes raw data to identify biological entities (e.g., genes) mentioned in captions.
Storage: Stores structured data in PostgreSQL with tables for Papers, Figures, and Entities. Optimized for fast querying via SQLAlchemy ORM.
Access: REST API (FastAPI): Securely query data with API key authentication. Supports JSONoutputs.
